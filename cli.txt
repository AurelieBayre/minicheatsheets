pwd
- print working directory

ls
- list
- shows all files and folders of current directory

ls -a
- list all files including hidden files

cd
- change directory
- go inside a folder
- cd <folder name>

 cd ..
 - go back one level

tab
- automatically completes filename

rm
- remove
- Eg. rm <filename>

rm -rf
- remove entire directory
- (rf)recursive force - remove child directories
- force delete everything inside the folder
- rm -rf <filename>
- rm -rf / - dangerous, will remove root directory

touch
- make a new file
- Eg. touch <filename>

touch filename filename
- create multiple files at once

mkdir
- make a new directory
- Eg. mkdir <foldername>

mkdir <filename> <filename>
- create 2 directories at once

rmdir
- Removes the directory.
- Eg. rmdir toBeRemoved

mv
- move and rename files
- Eg. mv <filename> <path/newname>

cp
- Copy files from one directory to another.
- Eg. cp new.txt /home/someDirectory/

man
- For getting the manual of a command.
- Eg. man cd

locate
- Used to Locate a file in Linux OS. Returns absolute path to that file. 
- Eg. locate index.html
